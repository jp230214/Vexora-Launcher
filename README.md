# Vexora-Launcher
The GitHub repo for my Launcher’s download. VEXORA

This is a Launcher that i've spent months of my time working on. It is Open-Source, but you can not use my code without giving Specific credits to me, and my channel.

The code Linked above does not include the dependencies required to run it. When i build my Launcher as a .exe, it automatically includes these dependencies, but if you are doing testing with this you must install Node.js with npm. Then in the folder run npm install, then npm start. This will run the Launchers source code directly.
